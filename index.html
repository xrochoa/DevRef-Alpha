GIT and GITHUB Reference

Areas:

Working Directory (WD), Staging Index (SI), Repository (Repo) and Stash

Command
Option
Arguments
Description
Example

git init 
Initializes a repository and tracking of current folder
git status 
Shows status of repository
git add 
Adds changes from working index to staging index
git commit 
Adds changes from staging index to repository
git diff 
Shows differences between working directory and staging index
git branch 
Shows list of existing branches
